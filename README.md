# swiper_fade

一款样式特殊小巧的移动端滑动插件。

## Features

- 支持手势滑动
- 支持顺逆时针
- 支持 `cubic-bezier()` 各类缓动函数

## Usage

引入 `swiper_fade.js` 及 `swiper_fade.css`

使用 Gulp 构建，可查看完整 Demo。

安装依赖：
```
$ npm install
```

运行：

```
$ npm start
```
